우리가 하는 일

우리는 단순한 고등학생이 아닙니다. 우리는 데이터를 다루고, 시간을 자동화하며, 사용자 경험을 설계하는 팀입니다. 보충 시간표조차 반복적으로 찾아보는 시대는 끝났습니다. 우리는 수업 시간표의 자동화를 실현했고, 더 나은 학습 환경을 직접 구축합니다. 이 프로젝트는 기술과 일상, 그리고 학생의 현실을 연결하는 정밀한 자동화 도구입니다. 우리가 만든 시스템은 단순한 편의를 넘어, 학생의 시간을 되찾는 도구입니다.

보충 수업 시간표 자동화

학생들이 날짜와 학번만 입력하면, 해당 날짜의 8교시·9교시 보충 수업 정보를 자동 제공.

JSON 기반 시간표 데이터 분석

과목, 반, 요일 등을 포함한 데이터를 JSON 형태로 구조화하고, 이를 파싱해 실시간 처리.

웹 페이지 제작 및 배포

Flask 서버를 활용하여 직관적이고 가벼운 웹 인터페이스 구현.

UI/UX 간소화 설계

누구나 쉽게 사용할 수 있도록 복잡한 입력 없이 “학번 + 날짜”만으로 동작하는 사용자 친화적 구조.

시간표 정확도 향상을 위한 알고리즘 개선

알파벳 코드에 따라 자동으로 과목과 반을 매칭하고, 오류를 최소화하는 방식으로 지속 개선.

학생을 위한 실용적 도구 개발

학업에 집중할 수 있도록 정보 접근 과정을 줄여주며, 반복적인 수작업을 대체하는 효율성 제공.
